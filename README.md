# VBank (Virtual Bank)

Учебный виртуальный банк на Java + PostgreSQL: пользователи, счета, переводы, история операций. Доступ через REST API и CLI.

## MVP
- Регистрация и авторизация
- Роли: Client / Operator / Admin
- Счета (1 счёт на валюту), балансы
- Переводы по номеру счёта
- Пополнение/снятие как заявки с подтверждением оператора
- История операций, аудит действий
- Справочник валют (CRUD)

## Стек и архитектура
- Java, PostgreSQL
- Backend: монолит, REST api/v1/*
- Клиент: CLI к API
